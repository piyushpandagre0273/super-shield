# Car Images Directory

Place your car images in this directory with the following names:

1. **hero-car-1.jpg** - Main hero image (red sports car) - Used on homepage
2. **car-2.jpg** - Black Porsche Panamera - Used in gallery
3. **car-3.jpg** - Lamborghini Aventador - Used in gallery  
4. **car-4.jpg** - Red sports car on coastal road - Used in gallery
5. **car-5.jpg** - Premium vehicle with Super Shield window film - Used in gallery
6. **car-6.jpg** - Luxury car with paint protection film - Used in gallery

## Image Requirements:
- **Format**: JPG or PNG
- **Recommended size**: 1920x1080px or larger
- **Aspect ratio**: 4:3 or 16:9 works best
- **File size**: Optimize images to under 500KB for faster loading

## How to add images:
1. Save your car images to this folder (`public/images/cars/`)
2. Name them exactly as listed above
3. The website will automatically display them

You can add more images to the gallery by updating `app/gallery/page.tsx` and adding more entries to the `galleryItems` array.
