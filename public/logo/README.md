# Logo Directory

## Add Your Logo Here

Place your Super Shield logo image in this directory.

### File Name:
**logo.png** (or **logo.jpg**)

### Recommended Specifications:
- **Format**: PNG (with transparent background) or JPG
- **Size**: 200x200px to 400x400px (square)
- **Background**: Transparent PNG works best for dark theme
- **File size**: Under 100KB for fast loading

### Supported Formats:
- `logo.png` ✅ (Recommended - supports transparency)
- `logo.jpg` ✅
- `logo.jpeg` ✅
- `logo.webp` ✅

### How to Add:
1. Save your logo file to this folder: `public/logo/`
2. Name it exactly: `logo.png` (or `logo.jpg`)
3. Refresh your browser - the logo will appear automatically in the navigation bar

### Note:
If you use a PNG with transparent background, it will look great on the dark theme. If you use JPG, make sure the background color matches your design.
